# AGENTS

## English (primary)
- Keep docs English-first; Chinese translations must be secondary and clearly labeled.
- Engine/runtime logs must be English only; do not add language-branching in engines.
- Run `make ci` before committing when possible.
- Prefer small, auditable changes; one logical change per commit.

## 中文（次要）
- 文档以英文为主；中文翻译为次要部分并明确标注。
- 引擎/运行日志仅使用英文，不要在引擎中做语言分支。
- 可行时在提交前运行 `make ci`。
- 变更尽量小且可审计；每次提交保持单一逻辑改动。
